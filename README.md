# Calculator Using DOM

A simple calculator built entirely with **HTML**, **CSS**, and **JavaScript**, where all calculator buttons (numbers and operators) are **dynamically created using the DOM API** (`document.createElement`).  
This project demonstrates DOM manipulation, event handling, and basic arithmetic operations without any external libraries.

---

## 🚀 Features
- **Dynamic Button Creation** — No hardcoded HTML buttons; all are generated via JavaScript.
- **Basic Arithmetic** — Supports addition, subtraction, multiplication, and division.
- **Clean Layout** — Styled with CSS for a simple calculator interface.
- **Reusable Functions** — Modular code for creating and appending elements.
- **NodeList to Array Conversion** — Easily manipulate created elements with array methods.
